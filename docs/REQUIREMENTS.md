# Beacon – Technical Requirements

This document outlines the technical framework and key dependencies for the Beacon Engagement Discovery Portal. It is not limited to Python and covers the full stack architecture.

---

## Web Framework
- Next.js (v14+)
- React (v18+)
- TypeScript (v5+)

## UI Framework
- Tailwind CSS (v3+)
- shadcn/ui (v1+)

## Backend API
- tRPC (v10+)

## Database
- PostgreSQL
- pgvector extension

## ORM
- Drizzle ORM (v1+)

## Authentication
- Azure AD SSO
- NextAuth.js (v4+)

## AI/LLM
- Anthropic Claude
- Azure OpenAI

## Email Service
- Azure Communication Services
- SendGrid

## Search Engine
- Meilisearch

## File Storage
- Azure Blob Storage

## Hosting
- Azure App Service

---

> This requirements list is for architectural reference and technology planning. It is not a Python package list.
